### 1 接口名称  

添加密钥接口    

### 2 接口描述    

- 添加访问密钥  
- 自定义用户名  
- 自定义过期时间  

### 3 请求地址  

`{apiAddress}`  

### 4 请求方式  

**GET**  

### 5 请求参数  

| 参数名    | 必选 | 类型   | 示例        | 说明     |
| --------- | ---- | ------ | --------------- | -------- |
| data   | 否   | string | 测试用户 | 用户名称 |
| expire  | 否   | string | 2077-11-11 20:30:01 | 过期时间 |

**注意事项**: 暂无   

### 6 请求示例

```
https://{apiAddress}/addkey.php?data=测试用户&expire=2077-11-11 20:30:01
```
